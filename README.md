探索游戏，开放（没有通关条件）

玩家控制主角，可以移动，交互，释放技能，视角跟随主角
目的探索，收集

平面地图，包括障碍物，可交互的npc和地形（例如门、宝箱）
生成：程序生成，可回溯
障碍物：
1. 墙（无法通过）
2. 减移速
可交互地形：
1. 门（钥匙通过npc获得）
2. 宝箱：直接增加属性

实体属性包括血量，防御，攻击，技能（被动技能，主动技能槽，后续改进加入技能，键位）
1. 血量
2. 防御：受到伤害为攻击/防御（后续改进）
3. 攻击：
    1. 近战攻击一定范围扇形120度，固定攻击速度
    2. 远程攻击，一定攻击距离，自动锁敌
2. 移动：固定移动速度
主角属性：
1. 移动：wasd控制移动方向
2. 攻击：近战攻击一定范围扇形120度，鼠标控制朝向，左键攻击
3. 背包：钥匙（后续加入物品如装备）

npc类型包括：
1. 友好：主角接近交互可以对话，提供给主角奖励
2. 敌对：
    1. 主动攻击主角
        1. 追击主角，和主角一定距离造成伤害
3. 中立：
    1. 对话触发决斗
    2. 被主角攻击之后会追击主角
基础状态：
1. 待机
2. 对话
3. 追击/攻击
4. 死亡，触发掉落物

交互：按钮触发，独立交互按钮不重叠

失败：血量归零时自动触发，于起点复活
